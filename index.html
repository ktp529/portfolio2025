Smart Visitor Access System

Description: This mobile app allows apartment residents to manage visitor access by filling out a visitor form and generating time-limited QR codes. Visitors can use the QR code to unlock the door, and residents can contact the guard house or management directly from the app.

Technologies Used:

  • Flutter
  • Dart
  • ESP32
  • Firebase (Cloud Server)
  • XAMPP Server (Local SQL Server) - Using PHP

Features:

  • Time-limited QR code generation for visitor access.
  • Real-time visitor tracking.
  • Integration with Firebase for storing visitor data and managing access.

User can login as resident or admin.

Admin can track real-time visitor requests.

Resident can contact management, guard house for help, or fill visitor form to request qr code to enter the resident area.

Enterprise Network Redundancy & VLAN Implementation

Description: Designed and deployed a fault-tolerant network infrastructure with HSRP redundancy and VLAN segmentation. Implemented cross-VLAN routing with 0% packet loss and validated failover mechanisms through controlled interface shutdown tests.

Technologies Used:

  • Cisco Packet Tracer
  • HSRP (Hot Standby Router Protocol)
  • VLAN configuration
  • Inter-VLAN Routing
  • Network Troubleshooting

Features:

  • Dual-router redundancy with HSRP preemption
  • VLAN segmentation (HR & Sales departments)
  • Cross-switch connectivity testing
  • Comprehensive failover validation

Network Architecture: Dual switches with VLAN 10/20 connected to redundant routers

HSRP Status: Active/Standby verification using show standby command

0% Packet Loss: Successful ping between PC0 (VLAN10) and PC2 (VLAN20)

HSRP Failover: Router1 transitioning from Standby to Active after Router0 shutdown
